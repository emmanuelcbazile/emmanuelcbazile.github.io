My Final Project

My program for my final project is a game that benefits the user by entertainment.
My game, Space Pirate Pilot, is designed for users to score points by collecting items.
In my game, collecting a star item will award you points, however, collecting an astroid
or enemy ship will deduct points. I created this game to increase in difficulty after the
user scores 10 points. When the user scores 10 points, an additional onject will project on
the game board which will deduct further points if collected. To play, use the space bar or
letter keys to move up and down the game board to collect stars and avoid other objects
